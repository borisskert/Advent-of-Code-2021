# Advent-of-Code-y2021

This is my code repository for [Advent of Code 2021](https://adventofcode.com/2021).

I use [Haskell Tool Stack](https://haskellstack.org) to build, test and run this project.

## Build

```shell
stack build
```

## Run tests
```shell
stack test
```

## Run exe
```shell
stack run
```
